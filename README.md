# CTU Cybersecurity Research Tool

## Colorado Technical University - Advanced Mobile Security Research

### Overview
This repository contains the CTU Cybersecurity Research Tool developed for **CS457: Advanced Mobile Security & Penetration Testing** at Colorado Technical University. The tool provides comprehensive analysis of mobile device security vulnerabilities for academic research purposes.

### Features
- Comprehensive phone number security analysis
- MAC address vulnerability assessment
- Carrier information lookup
- Network vulnerability identification
- Academic research documentation
